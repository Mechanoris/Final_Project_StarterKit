# SHDH2367 Interactivity Toolkit

Toolkit plugin for Unity 6 URP used in SHDH2367 Interactivity in Game Context.

## Features

- Tools menu dashboard and setup wizard.
- One-click setup for a playable scene in an empty/new scene.
- Three gameplay templates:
  - First Person
  - Third Person
  - Side View (level-style)
- Modular toggles:
  - Player
  - Game Loop
  - HUD
  - Collectibles
  - Feedback
- Optional sample level generation per gameplay mode.

## How To Use

1. Open Unity project with URP and Input System enabled.
2. Go to `Tools > SHDH2367 Toolkit > Dashboard` or `Setup Wizard`.
3. Select gameplay mode and modules.
4. Click `Setup Current Scene` (or finish wizard).
5. Press Play.

## Notes

- The toolkit uses `Assets/InputSystem_Actions.inputactions` for input mappings.
- Use the Dashboard validation section to check and auto-fix basic project setup.
